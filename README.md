# E2E автотесты для Pokemon Battle

Автотесты на **Cypress** с **Page Object Model** для сайта [pokemonbattle.ru](https://pokemonbattle.ru). Тестируют полный сценарий покупки аватара для тренера покемонов.

## 📋 Что тестируем

| № | Тест | Результат |
|---|------|-----------|
| 1 | Успешная покупка аватара (CVV 125) | ✅ Passed |
| 2 | Ошибка: недостаточно средств (CVV 300) | ✅ Passed |
| 3 | Ошибка: неверный код из SMS | ✅ Passed |

## 🛠 Стек

- **Cypress** — фреймворк для E2E тестирования
- **JavaScript** — язык тестов
- **Page Object Model** — паттерн организации кода
- **GitHub** — хранение проекта

## Запуск тестов
1. Скачать проект
2. Перейти в терминале в директорию проекта
3. Установить зависимости:
```
npm install
```

4. Запуск в интерактивном режиме (с окном браузера):
```
npx cypress open
```
5. Запуск в терминале:
```
npx cypress run
```
